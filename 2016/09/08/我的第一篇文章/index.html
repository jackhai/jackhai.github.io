搭建静态博客的笔记

发表于 2016-09-08   |  

1.注册github

注册地址

注册比较简单可以自行百度，已有帐号的可以忽略。

—— 需要注意repository的名字和SSH公钥的添加。

2.准备的安装环境

2.1安装git和Node.js

git官网下载

node.js官网下载

2.2安装hexo

安装之前必须保证你的电脑中已经有上述两个应用程序

hexo的安装

注意在安装的时候切换到你所建立的博客目录下面（我的F:/blog)

执行命令

$ npm install -g hexo

3.初始化博客

3.1初始化

文件夹（F：/blog）中右键打开Git Bash

$ hexo init

再执行

$ npm install

以上语句的执行，Hexo会自动在目标文件夹建立网站所需要的文件，你可以本地查看。

3.2启动服务

在（我的F:/blog）目录下执行

$ hexo server或者hexo s

在浏览器中打开 http://localhost:4000/，这时可以看到Hexo已为你生成了一篇blog。

你可以按Ctrl+C 停止Server

3.3建立自己的第一个博客

在blog目录下执行gitbash命令

(1)新建一篇文章

$ hexo new “我的第一篇文章”

会在/source/_post里自动生成了“我的第一篇文章.md”文件，之后新建的文章都将存放在此目录下.

(2)启动服务

$ hexo server

(3)生成静态网页

$ hexo generate或hexo g

(4)在浏览器输入http://localhost:4000 即可查看网站.

4.将blog部署到Github Pages

4.1找到blog目录下的配置文件_config.yml,用编辑器打开此文件

找到此文件中的deploy字段，按照以下配置

deploy:

type: git

repo: git@github.com:sunningcarryhaha/sunningcarryhaha.github.io.git

branch: master

4.2安装扩展

$ npm install hexo-deployer-git –save

4.3部署

$ hexo d

5.购买域名并且解析

  • 进万网进行购买
  • 购买之后进行域名绑定
  • 在public文件先新建CNAME文件，将域名写入
  • 传到github上面
